# Hajautetut ohjelmistot ja pilvipalvelut kurssin ensimmäinen harjoitustyö

Tämän harjoitustyön tarkoituksena oli harjoitella UDP ja TCP sokettien käyttöä Javan avulla. Tehtävänä oli tehdä sovellus, joka tarjoaa lukujen summauspalvelua erikseen määritellyn palvelimen käyttöön. Palvelin oli valmiiksi toteutettu ja sen yhteyteen on määritelty protokolla, jonka avulla summauspalvelua tarjoava sovellus voi ottaa siihen yhteyttä. Tarkempi tehtävänanto löytyy [tästä](Tehtävänanto).
